Политика конфиденциальности

Приложение «Радио Онлайн» · обновлено: июнь 2026

Настоящая Политика описывает, как приложение «Радио Онлайн» (далее — «Приложение») обращается с данными пользователя. Устанавливая и используя Приложение, вы соглашаетесь с условиями ниже.

1. Какие данные мы собираем

Приложение не требует регистрации и не собирает персональные данные: мы не запрашиваем имя, телефон, email, геолокацию или платёжную информацию для работы Приложения.

2. Данные, которые хранятся на устройстве

Для удобства Приложение сохраняет локально на вашем устройстве (и не передаёт нам):

  • список избранных станций;
  • историю прослушанных треков;
  • настройки (тема, вид списка, громкость, будильник, буферизация).

Эти данные удаляются при очистке данных Приложения или его удалении.

3. Доступ в интернет и сторонние сервисы

Приложение использует интернет, чтобы:

  • воспроизводить аудиопотоки напрямую с серверов радиостанций;
  • искать станции и получать ссылки на потоки через открытый каталог radio-browser.info;
  • загружать логотипы станций из общедоступных источников.

При обращении к этим сервисам им может становиться доступен ваш IP-адрес и стандартные технические данные запроса — это необходимо для самой передачи данных. Мы не контролируем политику этих сторонних сервисов.

4. Разрешения приложения

  • Интернет — для воспроизведения радио.
  • Уведомления и фоновая служба — для плеера и управления воспроизведением на экране блокировки.
  • Bluetooth — для корректной работы с наушниками и автоматической паузы при их отключении.
  • Будильник — для запуска радио в заданное время.

5. Реклама

В текущей версии реклама не показывается. В будущих версиях возможен показ рекламы через сторонние рекламные сети, которые могут использовать обезличенные идентификаторы устройства. Об этом будет сообщено в обновлённой версии Политики.

6. Дети

Приложение не предназначено для сбора данных детей и не собирает персональные данные ни у кого из пользователей.

7. Изменения

Мы можем обновлять эту Политику. Актуальная версия всегда доступна по этому адресу.
